The Power Query Arsenal

🪄 Chapter 1: The Magic of 1-Click Import (Data တင်သွင်းခြင်း)

💡 Technical ဆန်ဆန် မပြောတော့ပါဘူး။ နေ့စဉ် သင်က Salesman (၃၀၀) ရဲ့ ဖိုင်တွေကို ပေါင်းပြီး စာရင်းချုပ်ရတဲ့ Sales Manager သို့မဟုတ် Data Analyst တစ်ယောက် ဆိုပါစို့။ ဖိုင်တွေကို တစ်ခုချင်း ဖွင့်ပြီး Copy/Paste လုပ်ရတဲ့ ဒုက္ခက နေ့တစ်ဝက်လောက် အချိန်ကုန်ပါတယ်။

ဒါပေမဲ့ Power Query ကို သုံးလိုက်မယ်ဆိုရင် Folder တစ်ခုလုံးကို ဆွဲထည့်ပြီး "Refresh" တစ်ချက်နှိပ်ရုံနဲ့ စက္ကန့်ပိုင်းအတွင်း ဖိုင် ၃၀၀ လုံး Auto ပေါင်းသွားပါပြီ။ Excel Formula တွေ တစ်ကြောင်းမှ ရေးစရာ မလိုပါဘူး။

🛠️ ပါဝင်မည့် သင်ခန်းစာများ (Lessons 1.1 to 1.3)

  • 1.1 Importing Multiple Tables: Table အများကြီးကို တစ်ပြိုင်နက်တည်း ဆွဲယူခြင်း။
  • 1.2 Importing CSV: ကြီးမားတဲ့ CSV ဖိုင်တွေကို လွယ်ကူစွာ ကိုင်တွယ်ခြင်း။
  • 1.3 Folder Importing: Folder ထဲရှိ ဖိုင်အားလုံးကို တစ်ခါတည်း ပေါင်းစပ်ခြင်း (The Ultimate Time Saver)။

*(ပုံ - Folder ထဲရှိ ဖိုင်များအားလုံးကို Click တစ်ချက်ဖြင့် ပေါင်းစပ်နေပုံ)*

🤝 Chapter 2: The Data Matchmaker (ပေါင်းစပ်ခြင်းနှင့် ခွဲခြမ်းခြင်း)

💡 VLOOKUP တွေ ရေးပြီး Excel ကြီး Hang သွားဖူးပါသလား? Customer List နဲ့ Sales Record တွေကို တိုက်စစ်တဲ့အခါ (ဥပမာ - ဘယ် Customer က ဘာဝယ်သွားလဲ) Formula တွေ သောင်းဂဏန်းလောက် ရေးရတဲ့အခါ Excel ကြီး လည်သွားတတ်ပါတယ်။

Power Query ရဲ့ Merging စနစ်က Formula အစား Database Join စနစ်ကို သုံးထားလို့ Row သိန်းဂဏန်းရှိတဲ့ Data တွေကိုတောင် မျက်စိတစ်မှိတ်အတွင်း Error ကင်းကင်းနဲ့ ချိတ်ဆက်ပေးနိုင်ပါတယ်။ Accountant တွေ၊ Finance သမားတွေအတွက် လုံးဝကို သက်သာစေမယ့် အပိုင်းပါ။

🛠️ ပါဝင်မည့် သင်ခန်းစာများ (Lessons 2.1 to 2.4)

  • 2.1 Appending Data: လအလိုက် ခွဲနေတဲ့ Data တွေကို အောက်အောက်ချင်း ဆက်တိုက်ပေါင်းခြင်း။
  • 2.2 Merging Data: Table ၂ ခုကို ID တူရာရှာပြီး VLOOKUP ထက် မြန်ဆန်စွာ ချိတ်ဆက်ခြင်း။
  • 2.3 Conditional Column: Logic သုံးပြီး လိုချင်တဲ့ အဖြေကို IF Formula မလိုဘဲ အလွယ်တကူ ထုတ်ယူခြင်း။
  • 2.4 Customer Segmentation: ဝယ်သူတွေကို အတန်းအစားခွဲခြားပြီး Master Sales Table တည်ဆောက်ခြင်း။

🌪️ Chapter 3: Shaping the Chaos (ရှုပ်ထွေးနေသော Data များ ရှင်းလင်းခြင်း)

💡 System တွေက ထွက်လာတဲ့ Data တွေက ဖတ်လို့မရအောင် ရှုပ်ထွေးနေပါသလား? ERP (သို့) POS System ကနေ Export ထုတ်လိုက်တဲ့အခါ ခေါင်းစဉ်တွေလွဲ၊ Date တွေ မှား၊ ကွက်လပ် (Blank) တွေ အပြည့်ဖြစ်နေတဲ့ Data တွေကို ပြန်ရှင်းရတာ အလုပ်လုပ်ရတာထက် ပိုပင်ပန်းပါတယ်။

ဒီအခန်းမှာတော့ Data Analyst တွေ အသုံးအများဆုံး Data Cleaning Techniques (သန့်စင်ခြင်း နည်းလမ်းများ) ကို အသေးစိတ် သင်ကြားပေးသွားမှာဖြစ်လို့ Data ကြီး ဘယ်လောက် ပွစိတက်နေပါစေ ၅ မိနစ်အတွင်း သပ်ရပ်တဲ့ Master Table ကြီးတစ်ခု အဖြစ် ပြောင်းလဲပစ်နိုင်ပါပြီ။

🛠️ ပါဝင်မည့် သင်ခန်းစာများ (Lessons 3.1 to 4.5)

  • 3.1 Date Formatting: မှားနေသော Date များကို နိုင်ငံတကာစံနှုန်း ပြောင်းခြင်း။
  • 4.1 Fill Data: Blank ဖြစ်နေသော ကွက်လပ်များကို Auto ဖြည့်သွင်းခြင်း။
  • 4.2 Grouping: Category အလိုက် ပေါင်းချုပ် (Summarize) လုပ်ခြင်း။
  • 4.3 Extracting Text: လိုအပ်သော စာသားများသာ သီးသန့် ဆွဲထုတ်ခြင်း။
  • 4.4 Pivoting & 4.5 Unpivoting: Data Layout ကို မိမိလိုချင်သလို အလျားလိုက်၊ ဒေါင်လိုက် လှည့်ကစားခြင်း။

💼 Chapter 4: The Professional's Desk (လက်တွေ့ လုပ်ငန်းခွင် ပြဿနာများ)

💡 သင်ခန်းစာတွေ သိရုံနဲ့ မလုံလောက်ပါဘူး၊ လက်တွေ့ အသုံးချတတ်ဖို့က အရေးကြီးဆုံးပါ။ ဒါကြောင့် ဒီနောက်ဆုံးအခန်းမှာ လုပ်ငန်းခွင်မှာ တကယ်ကြုံရမယ့် "Real-world Scenario" ပုစ္ဆာကြီးတွေကို တိုက်ရိုက် ဖြေရှင်းပြသွားမှာပါ။

Accountant တွေ အမြဲခေါင်းစားရတဲ့ Bank Statement တွေကို စနစ်တကျ ပြန်စီတာ၊ လစဉ် Report တွေကို Folder လိုက်ပေါင်းပြီး Master Report ကြီး ဆွဲထုတ်တာတွေကို ဒီအခန်းမှာ သင်ယူရမှာ ဖြစ်ပါတယ်။

🛠️ ပါဝင်မည့် သင်ခန်းစာများ (Lessons 5.1 & 5.2)

  • 5.1 Real World (Bank Statement): ဘဏ်စာရင်းများကို System ထဲသွင်းနိုင်ရန် Auto Clean လုပ်ခြင်း။
  • 5.2 Real World (Multiple Files Folder): စာရင်းများစွာပါဝင်သော Folder ကြီးတစ်ခုလုံးကို ခလုတ်တစ်ချက်နှိပ်ရုံဖြင့် အဆုံးသတ် Report ထွက်သည်အထိ Automate လုပ်ခြင်း။
